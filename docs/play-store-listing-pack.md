## Full Description

Meet J.A.R.V.I.S Mobile, your personal voice-enabled AI assistant for Android.

J.A.R.V.I.S helps you interact naturally using voice or text, execute local actions, and connect with cloud AI models for advanced conversations.

Key features:

- Voice activation with speech-to-text input
- Natural assistant responses with text-to-speech playback
- Smart hybrid command routing (on-device actions + cloud AI)
- Fast chat-style interaction with response history
- Live backend integration for extensible workflows
- Clean futuristic interface inspired by Jarvis-style UX

Use J.A.R.V.I.S to:

- Ask questions and get AI-powered answers
- Open settings and trigger supported device actions
- Send commands through your connected backend agent stack

Permissions used:

- Microphone: for voice commands
- Optional device permissions: for call/message/navigation or requested actions
- Network access: to connect with your configured backend and AI services

J.A.R.V.I.S is designed for productivity, automation, and hands-free interaction.
